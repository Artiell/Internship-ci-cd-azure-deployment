Paraphrase T5

The Paragraph T5 model is a generating paraphrases model trained on the Google PAWS dataset. This modele will produce a paraphrase of the input text given.

Deep Sentiment

The Deep Sentiment is a model with an LSTM (Long Short Term Memory) neural architecture. This model will give in output a number between 0 and 1, where 1 is a positive sentence and 0 a negative sentence.

Keywords&Keyphrases

The Keyword & Keyphrases model is based on the KeyBert library. It leverages transformers and cosine similarity to efficiently extract keywords and keyphrases that better represent a given document. To use it you just have to give a text in input and submit it. The model will give you keywords and keyphrases from the input document.

Summarize Abstract

The goal of the Summarize Abstract model is to make summarys based on the input text given. It's an Abstractif model. The model will first analyse the input text then guess words to create a new summary with same mains topics as the original text.

Summarize Extract

As the Summarize Abstract model, the goal of the Summarize Extract model is to create summarys from the given input text. But it works differently. This model will extract keywords and keyphrases of the input text, then fill those with some sentences to create a summary.

Google T5

The Google T5 model is a Translation model. It takes English text only in input and can translate it in French, in Romanian and German. This model was developed at the origin by google
